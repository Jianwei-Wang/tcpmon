# TCPMon

TCPMon is a utility for viewing HTTP requests and responses. It is based on Apache's [TCPMon 1.0](http://ws.apache.org/tcpmon/).

See <http://robertbachmann.github.io/tcpmon/> for a tutorial.

## Change log
